Users

HTTP Methods Path Description
GET /user Retrieves all users from the database
GET /user/:id Retrieves a user using the user's ID (userID)
POST /user/Signup Signs up/Adds a new user to the database
PATCH/PUT /user/update/:id Updates/Edits/Alters user's details by ID (userID)
DELETE /user/delete/:id Deletes a user by their ID (userID)
POST /user/login/:id Logs a user in using their email (emailAdd)

Products

HTTP Methods Path Description
GET /product Retrieves all products from the database
GET /product/:id Retrieves a product using the product's ID (productID)
POST /product/add Add a new product to the database
PATCH /product/update/:id Update/Edits/Alters product's details by using it's ID (productID)

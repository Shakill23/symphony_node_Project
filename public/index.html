Users

HTTP Methods Path Description
GET /users Retrieves all users from the database
GET /users/:id Retrieves a user using the user's ID (userID)
POST /users/Signup Signs up/Adds a new user to the database
PATCH/PUT /users/update/:id Updates/Edits/Alters user's details by ID (userID)
DELETE /users/delete/:id Deletes a user by their ID (userID)
POST /users/login/:id Logs a user in using their email (emailAdd)

Products

HTTP Methods Path Description
GET /products Retrieves all products from the database
GET /products/:id Retrieves a product using the product's ID (productID)
POST /products/add Add a new product to the database
PATCH /products/update/:id Update/Edits/Alters product's details by using it's ID (productID)
